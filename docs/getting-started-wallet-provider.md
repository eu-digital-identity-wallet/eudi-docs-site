# Wallet Provider

EU Digital Identity Wallets will enable users to access online services, store and share digital documents, and create binding signatures. 
	
If you are a Wallet Provider (eg. Member State) and wish to learn more on how to use the Reference Implementation please follow the links below:

- for Android developers: [Android Wallet Provider](../eudi-app-android-wallet-ui/wiki/how_to_build/)
	
- for iOS developers: [iOS Wallet Provider](../eudi-app-ios-wallet-ui/wiki/how_to_build/)
	
- [iOS Wallet Provider](https://mgiakkou.github.io/devhub/eudi-app-ios-wallet-ui/wiki/how_to_build/)
